# macori.eu – Landing Page

Semplice pagina statica (HTML/CSS puro) pronta per GitHub Pages.

## Struttura
- `index.html` – contenuto
- `styles.css` – stile
- `CNAME` – dominio personalizzato (modifica con il tuo)
- `README.md` – istruzioni rapide

## Deploy rapido
1. Crea una repo pubblica su GitHub (es. `macori-landing` oppure `<username>.github.io`).
2. Carica i file, fai commit in `main`.
3. Imposta GitHub Pages: Settings → Pages → Source: `Deploy from a branch` → Branch: `main` (`/root`).
4. (Opzionale) Aggiungi dominio personalizzato nel campo `Custom domain` e abilita `Enforce HTTPS`.
